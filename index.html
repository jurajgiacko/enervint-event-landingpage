Program spolupráce

Tři úrovně podpory pro váš event

Úroveň přidělujeme na základě velikosti, disciplíny a strategické shody s Enervit.

Base

ENERVIT BASE

Pro vybrané eventy od 200 účastníků

Zvýhodněný nákup produktů Enervit, vzorky pro účastníky a brandingové materiály pro váš event.

  • ✓ Slevy na nákup produktů Enervit
  • ✓ Vzorky gelů a iontových nápojů
  • ✓ Bannery, vlajky a brandingové materiály
  • ✓ Stánky na občerstvovací stanice
  • ✓ Možnost komisního prodeje
Pro

ENERVIT PRO

Pro vybrané eventy od 500 účastníků

Iontový nápoj na občerstvovačky zdarma, sampling gelů a novinek pro účastníky, ceny do tomboly a kompletní zázemí.

  • ✓ Iontový nápoj na občerstvovací stanice zdarma
  • ✓ Rozdávačky gelů a novinek pro účastníky
  • ✓ Ceny do tomboly (produktové balíčky)
  • ✓ Stánky na občerstvovací stanice
  • ✓ Možnost komisního prodeje
  • ✓ Brandingové materiály a bannery
Exclusive

ENERVIT EXCLUSIVE

Na pozvání — TOP eventy nad 1 500 účastníků

Partnerství na míru nebo plný support. Produkty zdarma, kompletní zázemí a individuální dohoda.

  • ✓ Produkty zdarma pro event
  • ✓ Kompletní nutričný servis na míru
  • ✓ Plné zázemí (stánky, bannery, kelímky)
  • ✓ Společná PR a mediální spolupráce
  • ✓ Exkluzivita v kategorii sportovní výživa
  • ✓ Individuální smlouva — dohodneme se
Výběrový proces

Jak spolupráce funguje?

Přihlásíte event — my vyhodnotíme shodu a navrhneme úroveň spolupráce.

1

Přihlásíte event

Vyplňte formulář s detaily o vaší akci.

2

Posoudíme shodu

Vyhodnotíme váš event — disciplína, velikost, dosah. Odpovíme do 48 h.

3

Navrhneme balíček

Na základě profilu navrhneme BASE, PRO nebo EXCLUSIVE.

4

Doručíme a vy realizujete

Produkty dorazí 5 dní před akcí. Váš tým obsluhuje na místě.

5

Report a vyúčtování

Do 5 dní po eventu: fotky, feedback, vyúčtování.

Časté dotazy

Hodnotíme velikost (počet účastníků), sportovní disciplínu, mediální dosah, tradici a to, jak váš event ladí s naší strategií. Každou přihlášku posuzujeme individuálně.

Ano. Kapacita programu je omezená a ne každý event je pro nás vhodný. Pokud event nevybereme, dáme vám vědět s vysvětlením. Můžete se přihlásit znovu příští rok.

ENERVIT BASE je bezplatný. U ENERVIT PRO fungujeme na komisním modelu — vy prodáváte, 15 % z tržby si necháváte. ENERVIT EXCLUSIVE má individuální podmínky.

U balíčků BASE a PRO ne — dodáme vše potřebné a váš tým to na místě zrealizuje. Součástí PRO je video školení pro obsluhu stánku. Osobní účast řešíme pouze u EXCLUSIVE.

Alespoň 2 měsíce před plánovaným eventem. Kapacita programu je omezená a přihlášky vyhodnocujeme průběžně — čím dříve, tím lépe.

Fotodokumentaci (min. 5 fotek), vyúčtování komisního prodeje (pokud byl) a krátký feedback do 5 dní. Report je podmínkou pro schválení další spolupráce.
